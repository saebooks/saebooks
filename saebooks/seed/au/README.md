# Odoo l10n_au chart of accounts — reference copy

Copied from Odoo `l10n_au` module. Used to seed the default Australian
chart of accounts. These CSVs are stable reference data — vendored
rather than fetched at build time.

- `account.account-au.csv` — 124 accounts (AU CoA)
- `account.tax-au.csv` — AU GST + related tax codes
- `account.tax.group-au.csv` — tax groups
- `account.fiscal.position-au.csv` — fiscal positions
- `account.account.tag.csv` — account tags (BAS mapping etc.)
- `account.depreciation.model-au.csv` — depreciation templates

Upstream: https://github.com/odoo/odoo/tree/17.0/addons/l10n_au
